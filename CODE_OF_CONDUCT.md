# Code of Conduct

Porta Angusta is a small, friendly open-source project. Contributors are
expected to act with kindness, patience, and good faith toward one another.

## Our standards

We follow the **Contributor Covenant, version 2.1** as our Code of Conduct.
The full text is published at:

> https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: be welcoming, be respectful, assume good intent, and help others
feel safe contributing. Disagreement is fine; disrespect is not.

## Scope

This Code of Conduct applies anywhere Porta Angusta has a presence — this
repository, its issues and pull requests, email correspondence with
maintainers, and any other space where you are interacting with the
project or its community.

## Enforcement

Concerns can be reported privately to:

**onetracksystem@outlook.com**

Reports are read promptly and treated confidentially. The maintainer
(Nicholas F. Tominello) will review each report and respond with whatever
action is appropriate, which may include private feedback, a public
correction, a temporary or permanent ban from project spaces, or no action
if the report is found to be in bad faith.

We will follow the Contributor Covenant's
[Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines)
when deciding what action to take.
